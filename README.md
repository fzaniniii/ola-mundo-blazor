## 💡 Descrição

Este projeto é uma aplicação web desenvolvida com **Blazor utilizando .NET**, criada via terminal.

O objetivo da atividade foi dar os primeiros passos com aplicações web interativas, entendendo como funciona a renderização de componentes e a atualização dinâmica da interface.

---

## 🛠️ Comandos Utilizados

- `dotnet new blazor -n OlaMundoBlazor`  
  Cria a estrutura inicial do projeto Blazor.

- `cd OlaMundoBlazor`  
  Acessa a pasta do projeto.

- `code .`  
  Abre o projeto na IDE.

- `dotnet run`  
  Executa a aplicação web localmente.

---

## 📦 Estrutura do Projeto

Arquivos e pastas principais:

- `Pages/`  
  Contém as páginas da aplicação (ex: Home).

- `Shared/`  
  Componentes reutilizáveis.

- `Program.cs`  
  Configuração inicial da aplicação.

- `OlaMundoBlazor.csproj`  
  Arquivo de configuração do projeto.

Framework utilizado:

- `.NET 10.0 (Blazor)`

---

## ⚙️ Funcionalidade Implementada

Foi adicionada uma lógica de contador na página inicial da aplicação.

Diferente do comportamento padrão (incremento de 1), foi criado um ajuste para que o valor seja incrementado em **+250 a cada clique**.

---

## 🧠 Conceitos Praticados

- Componentização com Blazor  
- Interatividade com eventos (`onclick`)  
- Atualização dinâmica da interface  
- Manipulação de estado na UI
